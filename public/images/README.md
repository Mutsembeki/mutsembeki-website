# Pasta de Imagens

Coloque aqui as imagens estáticas do site:

- `artist.jpg` — Foto principal do artista para o Hero Section (recomendado: 800x800px ou maior, formato quadrado)
- `og-image.jpg` — Imagem para partilha em redes sociais (1200x630px)

Depois de adicionar `artist.jpg`, edite o ficheiro:
`src/components/sections/HeroSection.tsx`

E descomente o bloco `<Image>` indicado nos comentários, substituindo o placeholder.
